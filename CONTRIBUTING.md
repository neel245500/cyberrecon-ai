# Contributing

Thank you for considering contributing to CyberRecon AI. Please follow these steps:

1. Fork the repository.
2. Create a feature branch: `git checkout -b feat/awesome`
3. Run tests and linters locally (see `.github/workflows/python-ci.yml`).
4. Open a pull request with a clear description of changes.

For code style, follow existing project patterns. Keep PRs small and focused.
